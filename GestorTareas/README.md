# Gestor de Tareas

Aplicación de consola desarrollada en C# que permite gestionar tareas conectándose a una base de datos SQL Server.

## Tecnologías utilizadas
- C# / .NET 10
- SQL Server
- Microsoft.Data.SqlClient

## Funcionalidades
- Agregar tareas
- Ver todas las tareas
- Marcar tareas como completadas
- Eliminar tareas

## Cómo ejecutarlo
1. Clonar el repositorio
2. Crear la base de datos ejecutando el script SQL incluido
3. Modificar el connectionString en Program.cs con tu servidor
4. Ejecutar con `dotnet run`
