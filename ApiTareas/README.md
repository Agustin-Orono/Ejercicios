# API de Tareas

API REST desarrollada en C# con ASP.NET Core y Entity Framework conectada a SQL Server.

## Tecnologías utilizadas
- C# / .NET 10
- ASP.NET Core
- Entity Framework Core
- SQL Server

## Endpoints
- GET /api/tareas - obtener todas las tareas
- POST /api/tareas - crear una tarea nueva

## Cómo ejecutarlo
1. Clonar el repositorio
2. Crear la base de datos GestorTareas en SQL Server
3. Modificar el connectionString en appsettings.json
4. Ejecutar con dotnet run
